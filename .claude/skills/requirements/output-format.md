# 出力フォーマット（Jira コメント）

確定版は以下の形式で Jira チケットへコメントする。チケット本文は書き換えない。

```md
## 確定版 受入れ条件・品質条件（P2）

### 受入れ条件
- AC-1 🔵 [integration] {条件または操作} のとき、{期待結果} になる（出典: チケット本文）
- AC-2 🔵 [E2E] {条件} のとき、{期待結果} になる（出典: 2026-06-12 PdM 回答）

### 品質条件
- QC-1 🔵 [CI] {測定可能な基準}（例: 検索 API の p95 応答時間が現状から悪化しない）
- QC-2 🔵 [手動] {基準と確認方法}
- 他は対象外（性能・セキュリティ: 影響なしを確認済み）

### 備考
- {対象外にした項目とその理由、適用した Intent Delta}
```

ルール:

- ID は欠番を作らない。後から追加する場合は末尾に追記し、変更は新しいコメント（Delta）で表す
- マーカーは確定時点で全項目 🔵 であること（🟡・🔴 が残る場合は確定しない）
- 出典は人間が検証できる粒度で書く（「チケット本文」「{日付} の {誰} の回答」「既存仕様 {リンク}」）
- テストレベル `[unit|integration|E2E]` は AC に、検証手段タグ `[CI|テスト|手動]` は QC に付ける
