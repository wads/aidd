# Output Format

## Branch Review

```md
## レビュー結果

### ブランチ情報
- ブランチ: <ブランチ名>
- 変更ファイル数: N
- コミット数: N

### 把握したアーキテクチャ
<推定したレイヤー構成・責務分担の概要。不明な場合はその旨>

## 観点①：規約
### Critical
- <問題>

### Warning
- <問題>

## 観点②：設計
### Critical
- <問題>
  - 何が問題か
  - 改善提案

### Warning
- <問題>
  - 改善提案

### Suggestion
- <改善提案>

## 観点③：仕様適合
### Critical
- <問題>

### Warning
- <確認事項>

## 良い点
- <良い点>

## まとめ
<総合評価と優先アクション>
```

## PR Comment Review

```md
## PRレビューコメント一覧

**PR**: #<番号> <タイトル>
**レビュー状態**: APPROVED / CHANGES_REQUESTED / COMMENTED

### 対応必須
- <コメント>

### 要確認・質問
- <コメント>

### 任意対応
- <コメント>

### まとめ
- 対応必須: N件
- 要確認: N件
- 任意: N件

**推奨する次のアクション**:
1. ...
2. ...
```
